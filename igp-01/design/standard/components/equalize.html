Equal height (equalize) – Standard web pages

Use to make grid cells the same height. Make the inner elements (panels, wells and so forth) inherit the height from the parent grid.

<!-- Equalize nested elements -->
<div class="row wb-eqht"> 
			<div class="col-md-4">
			<section class="well hght-inhrt">
				<h3>Short container</h3>
<p>Example text.</p>
				</section>
				</div>
				<div class="col-md-4">
				<section class="well hght-inhrt">
				<h3>Medium container</h3>
<p>Example text.<br> Example text. </p>
				</section>
				</div>
				<div class="col-md-4">
				<section class="well hght-inhrt">
				<h3>Long container</h3>
<p>Example text. <br>Example text. <br>Example text. </p>
				</section></div>
</div>

Coding considerations:

  • The class hght-inhrt is applied on the element to be equalize as well as all the parents (up to the child of wb-eqht), whereas eqht-trgt is only applied to the element to equalize (nothing needs to be done about the parents).
