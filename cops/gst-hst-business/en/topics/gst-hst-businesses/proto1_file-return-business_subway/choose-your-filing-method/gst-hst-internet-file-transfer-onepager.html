GST/HST Internet File Transfer

GST/HST Internet File Transfer is an online service provided by the Canada Revenue Agency (CRA) that allows eligible businesses to file their GST/HST returns directly over the Internet. It leverages third-party accounting software to simplify the process. This service provides many advantages:

  • It meets the mandatory electronic filing requirement
  • It is free to use
  • It is available seven days a week
  • It is fast and easy, secure and confidential
  • It provides immediate confirmation that the CRA has received your return
  • It reduces processing time
  • It speeds up refunds (especially when combined with direct deposit)
  • It eliminates the need to mail paper returns

How it works

The process begins with generating an electronic version of your GST/HST return using CRA-approved software. This software creates a specific file format with a ".tax" extension. You then access the GST/HST Internet File Transfer service through the CRA website's "Ready to file" page. After attaching your ".tax" file, you review and confirm its accuracy before submitting it to the CRA. The CRA conducts a preliminary check of your return. A confirmation number indicates your return has been accepted for processing. Alternatively, the CRA will send an explanation of any errors requiring correction. You must retain this confirmation number for your records.

A four-digit access code is essential for using GST/HST Internet File Transfer.

Eligibility

GST/HST Internet File Transfer is available to most GST/HST registrants, excluding those with accounts that are administered by Revenu Québec. If you are a selected listed financial institution (SLFI) that reports both the GST/HST and QST, you must determine which returns to complete and when to file.

The following registrants are not eligible to use GST/HST Internet File Transfer and must use an alternate filing method:

  • Registrants whose accounts are administered by Revenu Québec
  • Builders required to report transitional information on housing (Schedule A of the GST/HST return)
    • These filers must use GST/HST NETFILE.
  • Registrants required to recapture input tax credits for the provincial portion of the HST on specified property or services (Schedules B and C of the GST/HST return)
    • These filers must use GST/HST NETFILE or file through a participating financial institution that offers electronic filing for GST/HST returns

Security and confidentiality

The CRA limits access to browsers supporting Transport Layer Security (TLS). TLS is a protocol that provides authentication, confidentially, and data integrity between communicating applications. Using the appropriate safeguards and configuration, TLS provides a method of transmitting data securely across networks. Effective implementation of TLS will assist in mitigating the most serious threats with web-based applications such as eavesdropping, data tampering, and message forgery. Upgrade to TLS 1.2 so that your system is as secure as possible.

First Nations Tax (FNT) accounts (Common section - move to choose method page?)

First Nations Tax (FNT) accounts can use GST/HST TELEFILE. However, after filing your return, you must mail Form GST 499-1 (the corresponding FNT schedule) to the Atlantic Tax Centre.

Filing a return with a rebate (Common section - move to choose method page?)

You can file returns with a rebate amount on line 111 using GST/HST Internet File Transfer. However, you must also mail the rebate application forms to the relevant tax centre on or before the due date of your return.

Joint filers (Common section - move to choose method page?)

If you are the coordinator of a closely related group of corporations (i.e. you have filed election GST303) and you file returns electronically, you must still send your letter of direction to your tax centre.

Retaining supporting documents (Common section - move to choose method page?)

You do not have to send receipts or supporting documents when you file your GST/HST information electronically. However, keep all supporting documents for six years, as the CRA may contact you for more information after they process your return.

You should also keep a working copy of your return for your records. It, and any other information you use to prepare your return, are subject to audit and must be retained for verification purposes.
