1. How do I use GST/HST Internet File Transfer?

If you are eligible to use GST/HST Internet File Transfer, access the screen through the Ready to file page and attach your .tax file. Once you have attached the file, you confirm that you want to file the information as your return. You will immediately get a confirmation number.

2. What are the benefits of using GST/HST Internet File Transfer?

GST/HST Internet File Transfer is fast, easy to use, and available seven days a week. The service reduces processing time and ends the need to mail a paper return, resulting in faster refunds.

3. Is GST/HST Internet File Transfer secure and confidential?

Yes. We use sophisticated encryption technology and security procedures to protect your personal information at all times. However, while we have made every possible effort to ensure the safety and integrity of transactions at our Web site, the Internet is a public network and therefore is outside our control.

4. Do I need to buy software or equipment to use GST/HST Internet File Transfer?

You will need to have accounting software that supports the GST/HST Internet File Transfer. Consult our list of certified software vendors. If you do not wish to use accounting software, you can still file your return electronically using GST/HST NETFILE.

5. Do I still have to send a paper return if I use GST/HST Internet File Transfer?

If you file your return using GST/HST Internet File Transfer, do not send us a paper copy of your return.

However, if you are including a rebate amount on your return you must mail the corresponding rebate application form(s) to the applicable tax centre for separate processing. The tax centre should receive the form(s) on or before the due date of your return.

After we have processed your return we may contact you for more information, so keep all supporting documents for six years. This applies whether you file on paper or electronically.

6. If I file a return using GST/HST Internet File Transfer, when does the CRA consider it to be received?

We consider a return that you file using GST/HST Internet File Transfer to be received as soon as we send you a confirmation number.
