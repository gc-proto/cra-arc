Note

In Quebec, Revenu Québec administers the GST/HST with the exception of selected listed financial institutions (SLFIs) that report both the GST/HST and QST. If you are an SLFI that reports both the GST/HST and QST, you can file your combined GST/HST and QST return (Form RC7200) using My Business Account, Represent a Client or GST/HST NETFILE.

For all other businesses located in Quebec, visit the Revenu Quebec web site.

GST/HST return

You can file the GST/HST return for your business electronically in various ways, depending on your needs and preferences. Here are the features of each method.

  • My Business Account/ Represent a Client
    • File your GST/HST return (Form GST34)
    • File your combined GST/HST and QST return (Form RC7200)
    • File any applicable schedules with your return
    • View and adjust previously filed returns
    • File and adjust certain rebates
    • View your account balance
    • Access other return types such as payroll, T2
    • Registration and userid/password needed
  • GST/HST NETFILE
    • File your GST/HST return (Form GST34)
    • File your combined GST/HST and QST return (Form RC7200)
    • File any applicable schedules with your return
    • File certain rebates
    • No userid/password needed
  • Electronic Data Interchange (EDI)
    • File your GST/HST return (Form GST34)
    • File certain schedules with your return
    • Allows filer to submit payment at the same time
    • Most financial institutions in Canada offer this service
    • Accessed from your financial institution's Internet banking (no CRA userid/password needed)
    • Does not allow for filing of GST/HST rebates
    • Does not allow for filing of the combined GST/HST and QST return (Form RC7200)
  • GST/HST TELEFILE
    • File your GST/HST return (Form GST34)
    • Does not allow for filing of GST/HST rebates or schedules
    • Does not allow for filing of the combined GST/HST and QST return (Form RC7200)
    • No userid/password needed
  • GST/HST Internet File Transfer
    • File your GST/HST return (Form GST34)
    • Requires software purchase from licensed vendor
    • Information taken directly from accounting software automatically - no re-keying required
    • Does not allow for filing of GST/HST rebates or schedules.
    • Does not allow for filing of the combined GST/HST and QST return (Form RC7200)

GST/HST rebates

There are six GST/HST rebates that you can file electronically using My Business Account or GST/HST NETFILE:

  1. Public service bodies’ rebate and GST self-government refund (GST66)
  2. General application for GST/HST rebates (GST189)
  3. New housing rebate for houses purchased from a builder (GST190)
  4. Rebate for federal, provincial and territorial governments (GST490)
  5. Rebate Application for Provincial Part of Harmonized Sales Tax (GST495)
  6. GST/HST New Residential Rental Property Rebate Application (GST524)

Also, non-profit organizations can file their annual government funding form (GST523-1) using My Business Account or the Non-profit organizations - government funding web form.

GST/HST elections

Almost all GST/HST elections can be filed electronically using My Business Account.
