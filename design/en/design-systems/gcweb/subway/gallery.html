Subway – GCWeb design system

GCWeb design system

Subway

The CRA is the creator of the subway template.

When to use

Break up long and complex tasks into pages that each focus on a step or specific answer people need before moving to the next step or section. Ideal for presenting services and processes.

Group related task steps together and provide logical navigation between the pages. Keep the number of main task steps to a reasonable number (aim to have no more than 8).

GC core version

The basic version, where there is an index page and one level of steps, is part of the GC core design system.

For this version, use:

  • Code for Subway main index page
  • Code for Subway content page

This version is in use for EI regular benefits.

CRA variant

The CRA is working on evolving the template and the governance, including when and how to use sub-steps. As we evolve the pattern, additional features are being added and that documentation will be provided here in the coming months.

Examples of the CRA variant include:

  • Disability tax credit (DTC)
  • Canada Dental Benefit
  • One-time top-up to the Canada Housing Benefit
  • Home office expenses for employees
