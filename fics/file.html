How to file

The following information details how to complete and file a Part XVIII (FATCA) or Part XIX (CRS) information return.

How to complete and file a Part XVIII or Part XIX Information Return

Under Part XVIII and XIX of the Income Tax Act, Canadian financial institutions are required to:

  • Apply due diligence procedures to identify reportable accounts; and
  • Annually report to the CRA the information required on a Part XVIII and/or Part XIX Information Return.

Steps

  1. Register for a U.S. identifier (For Part XVIII only)

    Canadian financial institutions who have to report under Part XVIII must register with the Internal Revenue Service (IRS).  Once registration is approved, the IRS will issue a Global Intermediary Identification Number (GIIN) which is a 19 character identification string which must be included on your Part XVIII Information Return.
    If you are a sponsor filing on behalf of a reporting financial institution you will require a separate GIIN that includes “SP” in the category code (e.g. 98Q96B.00000.SP.124).
    Please note, the GIIN may not be available for use by the IRS up to 15 days after it is issued.
    Refer to:

    • IRS FATCA registration

  2. Register for a Business Number (BN) RZ Program Account (For Part XVIII & XIX)

    In order to file a Part XVIII or Part XIX information return, you will need a Business Number (BN) RZ program account. 
    If you already have a BN RZ account you can choose to use that account or you may also create a BN RZ account dedicated to Part XVIII and Part XIX information returns.
    Refer to:

    • How to register for a business number or a Canada Revenue Agency Program Account

    Once registered, you will need access to the CRA's login services:

    • My Business Account
    • Represent a Client (for authorized representatives)

    If following the exchange of data with other countries there are errors or missing information that needs to be amended you will be contacted through the email address you provided on filing and informed that your list of notification of errors is available for download on the CRA login services. It is important that the individual listed as the contact person also has access to the CRA login services or that there is a process within the financial institution, for the contact person to advise someone who has access to the CRA login services.
    Refer to:

    • CRA login services

  3. File Your Return Electronically

    You must file your Part XVIII and Part XIX information returns electronically using the CRA Internet file transfer (XML) or Web Forms applications. To find out more on these applications, go to:

    • Filing Information Returns Electronically (T4/T5 and other types of returns)

    For both the Internet file transfer (XML) and the Web Forms applications, you will need a Web access code (WAC) to file your information returns over the Internet, unless you use these applications throughMy Business AccountorRepresent a Client.

    If you do not have aWACor you have misplaced it, go toOnline Web access code or call the e-Services Helpdesk at1-800-959-5525.
