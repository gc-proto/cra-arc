  • Under proposed legislation New administrative policy for reporting Employee security (stock) options

    • Reason for change: Postponement of legislative change
    • Effective date: Benefits provided from June 25, 2024 to December 31, 2024
    • Refer to: Employee security (stock) options and T4 slip – Information for employers
    Brief description

    Under the new CRA administrative policy, for the reporting of security options benefits and related deductions which occurred during the period from June 25, 2024 to December 31, 2024, if:

    • You have not updated your payroll system to use the new codes 90, 91 and 92: You can use code 38 to report the benefit, and report the deduction as 1/2 of the value reported in code 38 using either code 39 or 41
    • You have updated your payroll system to use the new codes 90, 91 and 92:
      • You can use the new code 90 to report the benefit, and report the deduction as 1/3 of the value reported in code 90 using either code 91 or 92
      • You can use the code 38 to report the benefit, and report the deduction as 1/2 of the value reported in code 38 using either code 39 or 41

    This policy was introduced to reduce the burden on employers and payroll service providers as a result of the proposed changes which were to take effect June 25, 2024, but that will now become effective on January 1, 2026.

    • Reason for change: Legislation and plain language
    • Effective date: Benefits provided on or after June 25, 2024
    • Refer to: Employee security (stock) options
    Brief description

    You, as the employer (the corporation or the mutual fund trust), may grant options to your employees to acquire securities of your corporation.

    Beginning June 25, 2024, under proposed legislation, the security options deductions under paragraphs 110(1)(d) and 110(1)(d.1) of the ITA will also be reduced to 1/3.

    As a result of this proposed change, your reporting obligations for employee security options for the 2024 calendar year and after have changed.

    • Updated codes

      • Code 38 – Security options benefits - Before June 25, 2024
      • Code 39 – Security options deduction – 110(1)(d) - Before June 25, 2024
      • Code 41 – Security options deduction – 110(1)(d.1) - Before June 25, 2024

    • New codes

      • Code 90 – Security options benefits - On or after June 25, 2024
      • Code 91 – Security options deduction – 110(1)(d) - On or after June 25, 2024
      • Code 92 – Security options deduction – 110(1)(d.1) - On or after June 25, 2024

    These new and updated codes must be used for T4 slips filed for calendar year 2024 and after to report taxable employee security (stock) options.
