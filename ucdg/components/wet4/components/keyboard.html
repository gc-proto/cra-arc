How to implement

Appearance

Press Shift

Press Ctrl

Press Enter

HTML code 
<p>Press <kbd>Shift</kbd></p>
<p>Press <kbd><abbr title="Control">Ctrl</abbr></kbd></p>
<p>Press <kbd>Enter</kbd></p>
Prototyping libraries

The following prototyping libraries are currently being developed and will be available to download once the CRA user-centred design guide is officially launched:

  • Axure library
  • Figma UI kit

Correct use

  • Use the exact spelling, spacing and sentence case of the keyboard text (for example, Ctrl)

Incorrect use

  • Do not use for menu names, or content text
