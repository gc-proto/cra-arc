Navigating through the CVITP tax software

Get an overview of the UFile CVITP tax preparation software!

UFile CVITP is customized exclusively for use by CVITP volunteers. When UFile CVITP becomes available for the current tax year, approved organizations receive an email containing the software activation keys and the instructions for installation.

To help with the installation process, watch the tutorial How to install the UFile CVITP software.

UFile CVITP software is intended solely for use by CVITP volunteers

  • Do not copy the software under any circumstances
  • Use the software only to complete returns for eligible individuals
  • Do not use the software for monetary gain

Disclaimer

This learning guide uses screenshots from prior versions of the UFile CVITP software. Consequently, the images may differ slightly from the current version of the UFile CVITP. The content is accurate and generally, the only difference will be the tax year being referenced. If significant changes to the current year’s software occur, a new screenshot will be published as soon as possible.
