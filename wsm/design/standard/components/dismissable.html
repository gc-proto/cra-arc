Content approach

Purpose of dismissable content

Make content dismissable by users.

Use when you want to allow the user to dismiss non-primary content that they only need to see once (e.g., advertisements, notices).

Do not use when the content is primary content.

How to implement

Basic dismissable content

  1. Add the class wb-dismissable to the element that contains the content you want to make dismissable.

The content can be closed by the user (x), and won't be visible when they return to visit the page.
