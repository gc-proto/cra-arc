Toggle – Standard web pages

Use to toggle elements between on and off states.

On this page

  • When to use
  • How to implement
  • Related components and functions

When to use

Use when

  • clicking an element should control the state of itself or other elements on the page (e.g. a button to open/close all expand/hide elements)
  • elements should change state when the page is printed (e.g. all expand/hide elements should be automatically opened for print)
  • accordion widget behaviour is needed

Do not use when

  • The element being toggled already supports the behaviour natively (e.g. expand/hide elements).

How to implement

For instructions on how to implement, refer to Toggle on GitHub.
