Correct use

  • Use opacity from 90% to 10% to change the opacity of a foreground element, and make the background visible through it
  • 100% opacity gives an image full visibility, much like the default setting. Use it when a fade in to 100%, or fade out from 100% is required
  • Ensure that the new lighter text colour still passes accessibility colour contrast requirements
    • The opacity affects the whole element and everything within it

Incorrect use

  • Do not use if the opacity makes the foreground content difficult to see or read
